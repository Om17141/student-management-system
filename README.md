# 🎓 Student Performance Management System

A Flask-based web application to manage and analyze student performance.

## 🚀 Features
- Add, Edit, Delete Students
- Search Functionality
- Dashboard Analytics
- Grade System (A, B, C)
- Pass/Fail Detection
- Student Detail View

## 🛠️ Tech Stack
- Flask (Backend)
- SQLite (Database)
- HTML + Bootstrap (Frontend)

## ▶️ How to Run
1. Clone the repository
2. Install dependencies: pip install flask
3. Run: python app.py
4. Open: http://127.0.0.1:5000/

## 📊 Project Description
This system helps teachers track student performance, identify toppers and weak students, and analyze class results using dashboard analytics.
